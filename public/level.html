悦动力等级积分制度

此等级划分及升级公式参考魔兽世界游戏

相关文章: http://siwen1985.blog.163.com/blog/static/435737472009111733050379/

等级共划分为40级，由四种属性值决定：体魄值，文学值，魔法值，贝币值。

总积分 = 体魄值 + 文学值 + 魔法值 + 贝币值(?) + 其他

升级所需积分计算公式：

积分 = ((2 × 等级) + 难度系数(等级)) × 基础积分值((等级) × 积分系数(等级)

计算出的积分舍去百位后的部分，为100的倍数。

难度系数(等级) 是一个附加的难度系数增量:

难度系数(等级) = 0                              (等级 < 10)
难度系数(等级) = 1                              (等级 >=10, < 20)
难度系数(等级) = 3                              (等级 >=20, < 30)
难度系数(等级) = 6                              (等级 >=30, < 35)

难度系数(等级) = 5 × (等级 - 33)           (等级 >= 35, < 40)

基础积分值(等级) 完成一周任务所得积分，作为升级的参考单位值：

基础积分(等级) = 45 + (5 × 等级)

积分系数(等级)  用于减少升级所需要的积分(增加或减少升级难多)：

积分系数(等级) = 1                                      (等级 <= 10)
积分系数(等级) = (1 - (等级 - 10) / 100)            (等级 >10, <30)
积分系数(等级) = 0.82                                   (等级 >=30, <40)

积分获取

发表运动日志： 体魄值  20 + 等级

发表一篇文章： 文学值  15 + 等级

玩游戏：           魔法值  10 + 等级

贝币交易：           ？        （这里不建议将贝币作为积分依据）

登录奖励：

每日登陆奖励包括贝币奖励和积分奖励两种，计算公式：

连续登录天数 x 比例系数 + 等级 x 等级系数 + 随机数

(以下为参考设定值，可根据实际情况进行调节)

贝币奖励：

连续1 - 7天登录                比例系数 = 1, 等级系数 = 0.5, 随机数 = (0, 等级]

连续7天以上登录                比例系数 = 1.5, 等级系数 = 1, 随机数 = (0, 等级 x 2]

积分奖励:

连续1 - 7天登录                比例系数 = 10, 等级系数 = 1, 随机数 = (0, 等级]

连续7天以上登录                比例系数 = 20, 等级系数 = 2, 随机数 = (0, 等级 x 10]
