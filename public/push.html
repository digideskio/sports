使用websocket进行服务器端数据推送

流程

身份确认

服务器推送数据结构

推送数据分类

message

article

wallet

system

status

Notice

流程

用户登录 ---> 建立websocket连接 ---> 身份确认 ---> 等待接收服务器推送数据

身份确认

request:

{

    “latitude”:1.2314,

    “longitude”:3.13123,

    "token": ""        // access token from login result

}

                

response:

{

    "userid": "" // token对应的用户

}

服务器推送数据结构

基本结构(所有推送数据都使用此结构)：

{

        "type": "message/article/wallet",

        "push": {

                "type": "",

                "pid": "",

                "from": "",        

                "to": "",

                "body":[

                        {

                                "type": "",

                                "content": ""

                        }

                ]

        },

        "time":14871182734

}

推送数据分类

message

聊天相关通知：聊天(chat)，群聊(groupchat)，关注(subscribe)，取消关注(unsubscribe)

        {

        "type": "message",

        "push": {

                "type": "chat/groupchat/subscribe/unsubscribe",

                "pid": "",        // message id

                "from": "",         // userid

                "to": "",                // userid or group id for groupchat

                "body": [

                        {

                                "type": "text/voice/image/video",

                                "content": "text/URL"

                        },

                        ...

                ]

        },

        "time":14871182734

}

article

文章主题相关通知：赞(thumb)，评论(comment), 专家点评(coach)

        {

        "type": "article",

        "push": {

                "type": "thumb/comment/coach",

                "pid": "",   // article id

                "from": "", // userid

                "to": "",        // userid

                "body": [                        // 评论内容

                        {

                                "type": "text/voice/image/video",

                                "content": "text/URL"

                        },

                        ...

                ]

        },

        "time":14871182734

}

wallet 

钱包通知: 转帐(tx), 打赏(reward)

转帐(tx)

        {

                "type": "wallet",

                "push": {

                        "type":"tx",

                        "pid": "",        // txid

                        "from": "",        // userid

                        "to": "",                // userid

                        "body": [

                                ...

                        ]

                },

                "time":14871182734

        }

        打赏(reward)

        {

                "type": "wallet",

                "push": {

                        "type":"reward",

                        "pid": "",        // article id

                        "from": "",        // userid

                        "to": "",                // userid

                        "body": [

                                ...

                        ]                        

                },

                "time":14871182734

        }

system

发送/接收心跳(sendheart/recvheart):

{

  “type”: “system”,

  “push”: {

    “type”: “sendheart/recvheart”,

    “from”: “userA”,

    “to”: “userB”,

    “body”:[

      {“type”: “userid”, “content”: “userA”},

      {“type”: ”record_id”, “content”: “55655ea481b6031167000005”}

    ]

  },

  "time":14871182734

}

约跑/应约(runshare/runshared):

{

  “type”: “system”,

  “push”: {

    “type”: “runshare/runshared”,

    “from”: “userA”,

    “to”: “userB”,

    “body”:[

      {“type”: ”record_id”, “content”: “55655ea481b6031167000005”},

      {“type”: “latlng”, “content”: “31.123412,131.1341114”},

      {“type”: “locaddr”, “content”: “Shanghai”},

      {“type”: “time”, “content”: “1311345213”},

      {“type”: “image”, “content”:”http://example.com/imgurl.jpg”}

    ]

  },

  "time":14871182734

}

约赞/应约(postshare/postshared):

{

  “type”: “system”,

  “push”: {

    “type”: “postshare/postshared”,

    “from”: “userA”,

    “to”: “userB”,

    “body”:[

      {“type”: ”article_id”, “content”: “55655ea481b6031167000005”}

    ]

  },

  "time":14871182734

}

约PK/应约(pkshare/pkshared):

{

  “type”: “system”,

  “push”: {

    “type”: “pkshare/pkshared”,

    “from”: “userA”,

    “to”: “userB”

  },

  "time":14871182734

}

status 

状态更新: 坐标(loc), 设备号(device), 用户状态(lock, ban, unban), 任务奖励(task)

坐标(loc):        

{

                “type”:”status”,

                “push”: {

                        “type”:”loc”,

                        “pid”:””,

                        “from”:””,

                        “to”:””,

                        “body”:[

        {

                “type”:”latlng”,

                “content”:”21.1211312,31.2131424”

},

{

        “type”:”locaddr”,

        “content”:”Shanghai”

}                

]

},

"time":14871182734

        }

设备号(device):        

{

                “type”:”status”,

                “push”: {

                        “type”:”device”,

                        “pid”:””,

                        “from”:””,

                        “to”:””,

                        “body”:[

        {

                “type”:”token”,

                “content”:”XXXXXXXX”

}                

]

},

"time":14871182734

        }

        用户状态(lock, ban, unban)：

{

                “type”:”status”,

                “push”: {

                        “type”:”lock/ban/unban”

                },

"time":14871182734

        }

        lock - 禁言， ban - 拉黑， unban - 正常

        

        任务奖励(task):        废弃

        {

                “type”:”status”,

                “push”: {

                        “type”:”task”

                        “body”:[

                                {

                                        “type”:”physique_value”,

                                        “content”: “30”

                                },

                                {

                                        “type”:”coin_value”,

                                        “content”:”3000000000”

                                }

                        ]

                },

"time":14871182734

        }

        

Notice

内部推送通知，由APP自身处理，不显示。

升级(levelup):

{

  “type”: ”notice”,

  "time": 14871182734,

  “push”: {

    “type”: ”levelup”,

    “to”: “userid”

  }

}

任务完成(taskdone):

{

  “type”: ”notice”,

  "time": 14871182734,

  “push”: {

    “type”: ”taskdone”,

    “to”: “userid”,

    “body”: [

      {“type”: ”physique_value”, “content”: “30”},

      {“type”: ”coin_value”, “content”: ”3000000000”}

    ]

  }

}

任务拒绝(taskfailure):

{

  “type”: ”notice”,

  "time": 14871182734,

  “push”: {

    “type”: ”taskfailure”,

    “to”: “userid”

  }

}
