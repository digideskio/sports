使用websocket进行服务器端数据推送

流程

身份确认

服务器推送数据结构

推送数据分类

message

article

wallet

status

流程

用户登录 ---> 建立websocket连接 ---> 身份确认 ---> 等待接收服务器推送数据

身份确认

request:

{

    “latitude”:1.2314,

    “longitude”:3.13123,

    "token": ""        // access token from login result

}

                

response:

{

    "userid": "" // token对应的用户

}

服务器推送数据结构

基本结构(所有推送数据都使用此结构)：

{

        "type": "message/article/wallet",

        "push": {

                "type": "",

                "pid": "",

                "from": "",        

                "to": "",

                "body":[

                        {

                                "type": "",

                                "content": ""

                        }

                ]

        },

        "time":14871182734

}

推送数据分类

message

聊天相关通知：聊天(chat)，群聊(groupchat)，关注(subscribe)，取消关注(unsubscribe)

        {

        "type": "message",

        "push": {

                "type": "chat/groupchat/subscribe/unsubscribe",

                "pid": "",        // message id

                "from": "",         // userid

                "to": "",                // userid or group id for groupchat

                "body": [

                        {

                                "type": "text/voice/image/video",

                                "content": "text/URL"

                        },

                        ...

                ]

        },

        "time":14871182734

}

article

文章主题相关通知：赞(thumb)，评论(comment)

        {

        "type": "article",

        "push": {

                "type": "thumb/comment",

                "pid": "",   // article id

                "from": "", // userid

                "to": "",        // userid

                "body": [                        // 评论内容

                        {

                                "type": "text/voice/image/video",

                                "content": "text/URL"

                        },

                        ...

                ]

        },

        "time":14871182734

}

wallet 

钱包通知: 转帐(tx), 打赏(reward)

转帐(tx)

        {

                "type": "wallet",

                "push": {

                        "type":"tx",

                        "pid": "",        // txid

                        "from": "",        // userid

                        "to": "",                // userid

                        "body": [

                                ...

                        ]

                },

                "time":14871182734

        }

        打赏(reward)

        {

                "type": "wallet",

                "push": {

                        "type":"reward",

                        "pid": "",        // txid

                        "from": "",        // userid

                        "to": "",                // userid

                        "body": [

                                ...

                        ]                        

                },

                "time":14871182734

        }

status 

状态更新: 坐标(loc), 设备号(device), 用户状态(lock, ban, unban)

坐标(loc):        

{

                “type”:”status”,

                “push”: {

                        “type”:”loc”,

                        “pid”:””,

                        “from”:””,

                        “to”:””,

                        “body”:[

        {

                “type”:”latlng”,

                “content”:”21.1211312,31.2131424”

},

{

        “type”:”locaddr”,

        “content”:”Shanghai”

}                

]

},

"time":14871182734

        }

设备号(device):        

{

                “type”:”status”,

                “push”: {

                        “type”:”device”,

                        “pid”:””,

                        “from”:””,

                        “to”:””,

                        “body”:[

        {

                “type”:”token”,

                “content”:”XXXXXXXX”

}                

]

},

"time":14871182734

        }

        用户状态(lock, ban, unban)：

{

                “type”:”status”,

                “push”: {

                        “type”:”lock/ban/unban”

                },

"time":14871182734

        }

        lock - 禁言， ban - 拉黑， unban - 正常
